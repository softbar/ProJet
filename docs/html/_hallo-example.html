Erstellt ein Skript mit dem Ident Ident, Namen Name, Inhalt Inhalt und der Position Postion, sofern der angegebene Ident nicht schon vorhanden ist. In beiden Fällen wird die ObjektID des erstellten oder bereits vorhanden Skripts zurückgegeben.

Parameters
string$IdentIdent des Skripts
string$NameName des Skripts
string$ContentInhalt, welcher in das Skript eingetragen werden soll. Default == "<?\n\n//Autogenerated script\n\n?>"
integer$PositionPosition im Objektbaum und somit auch in der Visualisierung. Default == 0
Note
Der Inhalt von Inhalt wird jedes mal überschrieben, sodass auch nach Änderungen immer der korrekte Inhalt garantiert werden kann. Für Skripte welche bearbeitet werden sollen, ist es empfohlen ein Konfigurationsformular mit SelectScript-Auswahl zu nutzen
